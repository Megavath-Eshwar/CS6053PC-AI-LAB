# Vacuum Agent

## Introduction
This project demonstrates a **Vacuum World problem** in Artificial Intelligence.  
A vacuum cleaner agent cleans dirty rooms based on what it senses.

---

## Description
- There are **two rooms**: A and B  
- Each room can be **Dirty** or **Clean**  
- The vacuum cleaner can:
  - Move left or right
  - Clean the current room

---

## Technology Used
- Python
- Google Colab

---

## How to Run
1. Open **Google Colab**
2. Paste the full Python code into one cell
3. Run the cell

---

## Agent Type
- Simple Reflex Agent

---

## Output
- Shows room status (Dirty / Clean)
- Highlights vacuum location
- Displays last action

---

## Conclusion
This program helps understand how an intelligent agent interacts with its environment in a simple way.
